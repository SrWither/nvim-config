<h1 align="center">Shortcuts and Commands</h1>

## Standard shortcuts

| Shortcuts  | Action |
| ------------- | ------------- |
| Ctrl + C  | Copy selection  |
| Ctrl + X  | Cut selection  |
| Ctrl + V  | Paste copied text |
| Ctrl + F  | Search in file |
| Ctrl + L  | Search and replace in file |
| Ctrl + Backspace | Delete word |
| Ctrl + S | Save file |
| Ctrl + Q | Close file |
